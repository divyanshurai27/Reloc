\# Reloc



City-to-city household combo marketplace



\## Overview

Reloc helps people resell household items when relocating to a new city.  

Sellers can list items as \*\*combos or individually\*\*, and buyers can browse, order, pay, and track delivery.



\## Key Features

\- Bundle (combo) selling for movers

\- 1-click fast exit selling

\- End-to-end logistics (pickup → inspection → delivery)

\- Secure authentication with roles (buyer/seller)

\- Cloud image uploads via Cloudinary

\- Order tracking and status updates

\- Reviews \& ratings for items/combos



\## Tech Stack

\- \*\*Frontend:\*\* Vite + React (Client app)

\- \*\*Backend:\*\* Node.js + Express

\- \*\*Database:\*\* MongoDB (Mongoose models)

\- \*\*Storage:\*\* Cloudinary for images

\- \*\*Deployment Ready:\*\* Frontend (Vercel/Netlify), Backend (Render/Railway)
